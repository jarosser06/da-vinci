Da Vinci Framework Documentation
==================================

**Da Vinci** is a framework for rapidly developing Python-based AWS Cloud Native applications.

The framework consists of two packages that work together:

- **da_vinci** - Core runtime library for application business logic
- **da_vinci-cdk** - CDK library for infrastructure declarations
