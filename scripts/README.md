# Scripts

Build and distribution automation for the da_vinci monorepo.

## Planned Scripts

### Version Management
- `bump-version.sh` - Synchronize version across both packages

### Build & Distribution
- `build.sh` - Build wheel files for both packages
- `distribute.sh` - Publish packages to S3 PyPI
- `validate.sh` - Pre-distribution validation
